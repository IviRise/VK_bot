# IviBot
**IviBot** — это учебный проект. Бот для социальной сети **ВКонтакте**, который умеет показывать погоду, давать случайные факты и вести статистику запросов.

---

## Быстрый старт
Эти инструкции помогут тебе запустить копию проекта на своём компьютере для разработки и тестирования.
### Необходимые условия
*   Установленный **Python 3.9** или выше.
*   Установленный **Git**.
*   Аккаунт **ВКонтакте** и созданное **сообщество** (группа) для бота.
### Установка и запуск

1.  **Клонируй репозиторий**
    ```bash
    git clone https://github.com/IviRise/VK_bot.git
    cd VK_bot
    ```
2. **Создай и активируй виртуальное окружение**
    ```bash
    python -m venv venv
    # Для Windows:
    .\venv\Scripts\activate
    # Для Linux/macOS:
    source venv/bin/activate
    ```
3. **Установи зависимости**
   ```bash
   pip install -r requirements.txt
   ```
4. **Настрой переменные окружения**
   - В корневой папке проекта создай файл `token.py`.
   - Вставь в него свой токен доступа сообщества ВКонтакте. Получить его можно в настройках твоего сообщества в разделе «Работа с API».
5. **Запусти бота**
   ```bash
   python main.py
   ```
   Увидишь сообщение `Бот запущен. Ожидаю сообщения...` — всё работает!

## Примеры использования
Напиши боту в личные сообщения твоего сообщества:

- `/weather` Москва — получить погоду в Москве.
- `/help` — вывести список всех доступных команд.
- `/fact` — получить случайный факт.
- Нажми на кнопки в чате после команды `/start`.

---

## Стек технологий

- Python 3.9+
- VK API (`vk_api`)
- Open-Meteo API — для получения погоды.
- Nominatim (OpenStreetMap) — для геокодирования.
